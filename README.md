### PlexMediaServer install playbook

---

To install PlexServer using this playbook:
You will need ansible installed on a server and a target server. Note this playbook was designed for CentOS7 and CentOS8

```html
git clone https://github.com/wickedtechst/pmsinstall
cd pmsinstall
Update the inventory file and replace 192.168.1.171 with the your IP

ansible-playbook installplex.yml -i inventory -u root -k
```
